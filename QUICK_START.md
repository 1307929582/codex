# 🚀 一键部署（零配置）

## 真正的一键部署

```bash
git clone https://github.com/1307929582/codex.git
cd codex
./deploy-auto.sh
```

就这么简单！脚本会自动：
- ✅ 生成安全的数据库密码
- ✅ 生成安全的JWT密钥
- ✅ 构建Docker镜像
- ✅ 启动所有服务
- ✅ 创建管理员账户（可选）
- ✅ 显示访问地址

## 部署后

1. 访问 `http://localhost:3000/admin`
2. 使用创建的管理员账户登录
3. 在"系统设置"中配置OpenAI API密钥
4. 开始使用！

## 无需配置

- ❌ 不需要手动编辑.env文件
- ❌ 不需要手动生成密码
- ❌ 不需要手动创建管理员
- ❌ 不需要手动配置数据库

所有配置自动完成！
